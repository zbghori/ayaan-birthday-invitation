## 🚀 How to Publish on GitHub Pages

1. Create a new repository on GitHub named `ayaan-birthday-invitation`
2. Upload both files (`index.html` and `README.md`) to the repository
3. Go to **Settings** → **Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**
7. Wait 1–2 minutes — your invitation will be live at:
   ```
   https://YOUR_GITHUB_USERNAME.github.io/ayaan-birthday-invitation/
   ```
